Approach using lambda abstractions. 

- `LAMBDA-DEF.maude`: Definition of lambda abstractions
- `LAMBDA-MAP.maude`: Definition of higher-order `map` based on `LAMBDA-DEF` and `COLLECTIONS`
- `LAMBDA-FOLD.maude`: Definition of higher-order `fold` based on `LAMBDA-DEF` and `COLLECTIONS`. This file shows how to define `map` using `fold`. 
- `CURRIED-EXE.maude`: Example of curried functions
- `MAP-EXE.maude`: Examples using `map`.
- `FOLD-EXE.maude`: Examples using `fold`.
